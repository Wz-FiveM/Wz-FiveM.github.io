# Spawn Veh
```lua
NCS.spawnVehicle(model, position, heading, cb, inNetwork)
```

Exemple
```lua
NCS.spawnVehicle("sultan", vector3(0,0,0), 150.0, function(vehicle)
    FreezeEntityPosition(vehicle, true)
end)
```

| Argument      | Type            |
|---------------|-----------------|
| model ou Hash | string / nombre |
| position      | table           |
| heading       | nombre          |
| cb            | function        |
| inNetwork     | oui / non       |
